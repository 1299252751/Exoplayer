    Clock

    public final class AutoAdvancingFakeClock
extends FakeClock
    FakeClock extension which automatically advances time whenever an empty message is enqueued at a future time. The clock time is advanced to the time of the message. Only the first Handler sending messages at a future time will be allowed to advance time to ensure there is only one "time master". This should usually be the Handler of the internal playback loop.
